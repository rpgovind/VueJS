#Directives
## Vue.js directives are similar to angular directives

> v-if - ng-if <br/>
> v-else <br/>
> v-show -  ng-show <br/>
> v-for  - ng-repeat <br/>
> v-once  - one time binding <br/>
> v-pre <br/>
> v-cloak - ng-cloak  <br/>
> v-bind  eg:  v-bind:src , v-bind:alt can use :src  or :alt alternatively


## two-way binding
- v-model : for two way binding
